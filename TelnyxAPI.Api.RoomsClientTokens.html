    create_room_client_token(connection, room_id, create_room_client_token_request, opts \\ [])

    Create Client Token to join a room. Synchronously create an Client Token to join a Room. Client Token is necessary to join a Telnyx Room. Client Token will expire after token_ttl_secs, a Refresh Token is also provided to refresh a Client Token, the Refresh Token expires after refresh_token_ttl_secs.
