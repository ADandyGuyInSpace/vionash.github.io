    building

     Building

    To install the required dependencies and to build the elixir project, run:

    mix local.hex --force
mix do deps.get, compile

     Installation

    (currently not available in Hex, INSTALL MANUALLY FROM ABOVE)

    If available in Hex, the package can be installed by adding telnyx_api to your list of dependencies in mix.exs:

    def deps do
  [{:telnyx_api, "~> 1.0.0"}]
end

    Documentation can be generated with ExDoc and published on HexDocs. Once published, the docs can be found at https://hexdocs.pm/telnyx_api.
